## JavaScript

- **[Array Explorer](https://sdras.github.io/array-explorer/)** — Find the array method you need without digging through the docs.
- **[Babel](https://babeljs.io/repl)** — Use next generation JavaScript, today.
- **[Closure Compiler Service](https://closure-compiler.appspot.com/)** — Minifies JavaScript and estimates weight after GZIP compression.
- **[ESLint](https://eslint.org/demo/)** — A pluggable linting utility for JavaScript and JSX.
- **[JavaScript functions visualized](https://poet.codes/e/9YgdQVVD7Ip/frame/900px/515px)** — A JavaScript functions visualized example inspired by Wes Bos's tweet.
- **[JavaScript Visualizer](https://tylermcginnis.com/javascript-visualizer/)** — A tool for visualizing Execution Context, Hoisting, Closures, and Scopes in JavaScript.
- **[JS Tips](http://www.jstips.co/)** — JS Tips is a collection of useful daily JavaScript tips that will allow you to improve your code writing.
- **[JSNICE](http://jsnice.org/)** — Statistical renaming, type inference and deobfuscation.
- **[Object Explorer](https://sdras.github.io/object-explorer/)** — Find the object method you need without digging through the docs.
- **[plainJS](https://plainjs.com/)** — JavaScript functions and helpers.
- **[Regex101](https://regex101.com/)** — Online regex tester, debugger with highlighting for JavaScript.
- **[regexly](https://regexly.chipto.io/)** — A test editor for your regular expressions.
- **[The Vanilla JS Toolkit - Helper Functions](https://vanillajstoolkit.com/helpers/)** — Helper functions make complicated or repetitive tasks a bit easier, and keep your code DRY (an acronym for Don’t Repeat Yourself).
